# Regioni - Interactive Italian Wine Region Explorer

Discover the rich diversity of Italian wines through an interactive map showcasing all 20 wine regions of Italy.

## Features

- **Interactive SVG Map**: Click on any of Italy's 20 wine regions to explore their signature wines
- **60+ Italian Wines**: Comprehensive collection featuring 2-3 signature wines from each region
- **Detailed Wine Information**: Each wine includes grape variety, tasting notes, color classification, and fascinating facts
- **Random Discovery**: Explore Italian wines serendipitously with the random discovery feature
- **Smooth Animations**: Beautiful CSS animations for an elegant user experience
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **No Build Process**: Pure HTML, CSS, and JavaScript - just open and enjoy

## Wine Regions Covered

### Northern Italy
- Valle d'Aosta
- Piedmont (Barolo, Barbaresco, Moscato d'Asti)
- Liguria
- Lombardy (Franciacorta, Valtellina)
- Trentino-Alto Adige
- Veneto (Amarone, Prosecco, Soave)
- Friuli-Venezia Giulia
- Emilia-Romagna (Lambrusco)

### Central Italy
- Tuscany (Brunello, Chianti, Vernaccia)
- Umbria (Sagrantino di Montefalco)
- Marche (Verdicchio)
- Lazio (Frascati)
- Abruzzo (Montepulciano d'Abruzzo)
- Molise

### Southern Italy & Islands
- Campania (Taurasi, Fiano di Avellino)
- Basilicata (Aglianico del Vulture)
- Puglia (Primitivo)
- Calabria
- Sicily (Nero d'Avola, Etna Rosso)
- Sardinia (Cannonau, Vermentino)

## Local Development

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start exploring Italian wines!

No build process, dependencies, or installation required.

## GitHub Pages Deployment

1. Create a new repository on GitHub
2. Push this code to the repository:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Italian wines explorer"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/italian-wines-app.git
   git push -u origin main
   ```
3. Go to repository Settings ‚Üí Pages
4. Under "Source", select branch `main` and click Save
5. Your site will be live at `https://YOUR_USERNAME.github.io/italian-wines-app/`
